# Cursor Prompt 备忘录

## 写完文档之后，同步到线上文档

**使用场景**: 当你完成了文档编写或修改后，需要确保 Docusaurus 线上文档与本地代码保持同步

**Prompt**：

```
帮我同步所有的 /docs 下的文档和 /README.md 到 tools/docs 中，使得docusaurus的文档反映了最新的状态
```
